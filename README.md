This is the hardware repository for the Zaparoo project. It includes the assets required for completing Zaparoo hardware projects such as 3D models, PCBs and firmware.

## Project Index

Each folder in this repository represents a different project: 

- [diy-reader/](/diy-reader) - The [DIY Reader](https://zaparoo.org/docs/community-projects/diy-reader/) is a custom NFC reader built using a PN532 module.
- [pcb-cards/](/pcb-cards) - Create [PCB Cards](https://zaparoo.org/docs/tokens/pcb-cards/) for use as Zaparoo cards.
- [pn532-usbc-pcb/](/pn532-usbc-pcb) - A model of the [PN532 USB](https://zaparoo.org/docs/readers/nfc/pn532-usb/). Use this to help model custom cases.
- [pn532-usbc-case/](/pn532-usbc-case) - A simple case for the [PN532 USB](https://zaparoo.org/docs/readers/nfc/pn532-usb/)- the recommended ready-to-use reader for Zaparoo.
- [smes/](/smes) - The [Super Zap Boy](https://zaparoo.org/docs/community-projects/super-zap-boy/) is a custom case project compatible with the MiSTer and Zaparoo.
